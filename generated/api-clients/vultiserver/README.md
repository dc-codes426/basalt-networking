# Rust API client for basalt-vultiserver-client

TSS (Threshold Signature Scheme) server for keygen, keysign, and reshare
operations supporting ECDSA, EdDSA, and MLDSA via GG20 and DKLS protocols.

Vultiserver runs an Echo HTTP server and an asynq worker for async TSS
operations backed by Redis. Most mutating endpoints enqueue an async task
and return immediately.
